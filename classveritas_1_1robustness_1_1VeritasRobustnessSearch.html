def veritas.robustness.VeritasRobustnessSearch.get_max_output_difference (   self,
  delta 
)

Return tuple.

  • [0] max_output_diff: (max. possible output of target_at)

    • (min. possible output of source_at) If this value is <= 0, then it is not possible for target_at to be more confident than source_at of its prediction -> proof that no adv. exists for this delta

    else, if this value is > 0, then it may be possible (it is a bound after all) that a counter adversarial example exists

  • [1] list of generated (counter) examples for this delta, empty list if none found. If available, this is used to update the current value of upper, the upper bound on the minimal distance to the closest adversarial example
